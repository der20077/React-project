# 📝 TaskList (классовый компонент)

Простой компонент для управления списком задач, написанный на классовых компонентах React.

## 🧩 Функционал

- ✅ Добавление задач через текстовое поле
- ✅ Отображение списка задач
- ✅ Очистка поля после добавления

## 🛠️ Технологии

- React (классовые компоненты)
- JavaScript (ES6)

## 🚀 Использование

```jsx
import TaskList from "./components/TaskList";

function App() {
  return <TaskList />;
}
