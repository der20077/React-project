# 🔄 Lifting State Up (подъём состояния)

Демонстрация передачи состояния от родительского компонента к дочернему и обратно через пропсы.

Проект показывает, как:
- родительский компонент хранит состояние
- передаёт его в дочерний через пропсы
- дочерний компонент изменяет состояние через переданную функцию

---

## 🧩 Компоненты

### `ParentComponent`
- Хранит состояние `count`
- Передаёт `count` и функцию `increment` в дочерний компонент

### `ChildrenComponent`
- Получает `count` и `onIncrement` через пропсы
- Отображает текущее значение
- При клике вызывает `onIncrement`, обновляя состояние в родителе
